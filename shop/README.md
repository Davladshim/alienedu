# Магазин презентаций (модуль AlienEdu)

## Это отдельный модуль, который ведётся из другого чата/аккаунта Claude

**Перед началом любой работы здесь:**
1. Сделай `git pull`, чтобы получить последние изменения
2. Открой `/database/schema.sql` (в корне репозитория, на уровень выше этой папки) — это ЕДИНЫЙ источник правды о структуре базы данных
3. Если структура таблиц `presentations` или `access_codes` не совпадает с тем, что ты помнишь — значит платформа могла внести изменения. Свериться перед продолжением работы.

**Если меняешь структуру базы данных:**
1. Сначала обнови `/database/schema.sql`
2. Закоммить и запушь именно это изменение отдельно, с понятным сообщением коммита (например: "shop: добавлено поле discount_percent в presentations")
3. Только после этого продолжай работу с кодом, который зависит от этого изменения
